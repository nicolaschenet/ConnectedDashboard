ConnectedDashboard
=======

Just a sandbox for now ;)

## Install & Run

```bash
git clone git@github.com:nicolaschenet/ConnectedDashboard.git

cd ConnectedDashboard

node app.js

or

supervizor app.js
```

## View it in your browser

Simply type http://localhost:1337
(no need for apache running)
